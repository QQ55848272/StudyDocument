# 报表效果展示

## 轮播图效果

```js
var re=_g().getWidgetByName("report0"); //获取报表块对象
setInterval(function(){
	if(re.currentPage==re.totalPage){
		re.gotoPage(1) //当前页码等于总页数时，跳转到第一页
		}
	else{
		re.gotoNextPage()  //跳转到下一页
		}
	},3000)   //每 3 秒调
	
```
