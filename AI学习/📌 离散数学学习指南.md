## **📌 离散数学学习指南**

离散数学是计算机科学、人工智能和机器学习的核心数学基础之一，涵盖**集合论、图论、逻辑、组合数学、数理逻辑**等内容。以下是完整的**核心知识点、学习资源推荐、实践方法**，适用于不同阶段的学习者。

------

## **📚 核心知识点**

### **1. 逻辑与命题（Logic & Propositional Calculus）**

✅ **命题逻辑（Propositional Logic）**：真值表、逻辑运算（与、或、非、蕴含）
✅ **谓词逻辑（Predicate Logic）**：全称量词、存在量词
✅ **布尔代数（Boolean Algebra）**：

- 布尔变量、逻辑门
- 逻辑表达式的化简（卡诺图 Karnaugh Map）
  ✅ **自动推理（Automated Reasoning）**：用于人工智能中的**知识推理**

### **2. 集合论（Set Theory）**

✅ **集合运算**：交集、并集、补集
✅ **幂集（Power Set）**：所有子集的集合
✅ **笛卡尔积（Cartesian Product）**：用于数据库关系模型
✅ **关系（Relations）**：偏序关系、等价关系

### **3. 组合数学（Combinatorics）**

✅ **排列与组合（Permutations & Combinations）**
✅ **容斥原理（Inclusion-Exclusion Principle）**
✅ **鸽笼原理（Pigeonhole Principle）**：用于哈希函数分析
✅ **递归（Recursion） & 递归关系（Recurrence Relations）**
✅ **生成函数（Generating Functions）**：用于分析算法的时间复杂度

### **4. 图论（Graph Theory）**

✅ **图的基本概念**：无向图、有向图、加权图
✅ **遍历算法**：

- 深度优先搜索（DFS）
- 广度优先搜索（BFS）
  ✅ **最短路径算法**：
- Dijkstra 算法（单源最短路径）
- Floyd-Warshall 算法（全源最短路径）
  ✅ **最小生成树（Minimum Spanning Tree, MST）**：
- Prim 算法
- Kruskal 算法
  ✅ **图的应用**：
- **社交网络分析**
- **计算机网络（路由算法）**
- **人工智能（知识图谱）**

### **5. 关系代数 & 数据库（Relational Algebra & Databases）**

✅ **关系代数**：选择、投影、连接、并、交、差
✅ **SQL 查询的数学基础**
✅ **主键、外键、范式化**

### **6. 计算理论（Theory of Computation）**

✅ **有限自动机（Finite Automata）**：

- 确定性有限自动机（DFA）
- 非确定性有限自动机（NFA）
  ✅ **图灵机（Turing Machine）**：计算理论基础
  ✅ **计算复杂性（Computational Complexity）**：
- P vs NP 问题
- NP 完全性（NP-Completeness）

## **💻 实践练习**

### **1. 使用 Python 进行离散数学计算**

📌 **推荐工具库**：

- `networkx`（图论分析）
- `sympy`（符号计算）
- `itertools`（组合数学）

📌 **示例代码**：
📍 **生成排列和组合**

```
python复制编辑import itertools

# 生成排列
perm = list(itertools.permutations([1, 2, 3]))
print("排列:", perm)

# 生成组合
comb = list(itertools.combinations([1, 2, 3, 4], 2))
print("组合:", comb)
```

📍 **图的最短路径（Dijkstra 算法）**

```
python复制编辑import networkx as nx

G = nx.Graph()
G.add_weighted_edges_from([(1, 2, 1), (2, 3, 2), (1, 3, 4)])

shortest_path = nx.dijkstra_path(G, source=1, target=3)
print("最短路径:", shortest_path)
```

📍 **布尔代数计算**

```
python复制编辑from sympy import symbols, And, Or, Not

A, B = symbols('A B')
expr = And(Or(A, B), Not(A))
print("逻辑表达式:", expr.simplify())
```
