# 概率论与统计学

## **📌 为什么学习概率论与统计学？**

概率统计在 AI/ML 中有广泛应用，例如：

- **贝叶斯定理**：在**自然语言处理（NLP）**、**垃圾邮件过滤**中应用广泛。
- **最大似然估计（MLE）**：用于训练机器学习模型（如**逻辑回归**）。
- **蒙特卡洛方法**：在**强化学习**和**深度学习优化**中常见。
- **假设检验与P值**：在**A/B测试**和**数据科学实验**中非常重要。

------

## **📚 核心知识点**

### **1. 概率论基础**

✅ **基本概念**：随机变量、样本空间、事件、条件概率
✅ **贝叶斯定理**：用于贝叶斯网络、NLP 任务
✅ **常见概率分布**：

- 离散分布

  ：

  - **伯努利分布（Bernoulli）**：二分类任务（如抛硬币）
  - **二项分布（Binomial）**：多次独立实验（如广告点击预测）
  - **泊松分布（Poisson）**：罕见事件（如服务器请求数）

- 连续分布

  ：

  - **正态分布（Gaussian）**：许多自然现象符合正态分布
  - **指数分布（Exponential）**：用于计算事件发生时间（如排队系统）
  - **均匀分布（Uniform）**：所有结果等概率

### **2. 统计学基础**

✅ **期望值、方差、协方差、相关性**
✅ **大数定律（Law of Large Numbers）**
✅ **中心极限定理（Central Limit Theorem, CLT）**
✅ **最大似然估计（MLE） & 最大后验估计（MAP）**
✅ **参数估计（点估计 & 区间估计）**
✅ **假设检验（Hypothesis Testing）**：

- t 检验（T-test）：用于小样本均值比较
- 卡方检验（Chi-square test）：用于分类变量的独立性检验
- P 值（P-value）：衡量结果是否具有统计显著性 ✅ **回归分析（Regression Analysis）**：
- 线性回归
- 逻辑回归（Logistic Regression）

### **3. 机器学习中的应用**

✅ **贝叶斯分类器（Naive Bayes）**：垃圾邮件分类
✅ **马尔可夫链（Markov Chain）**：自然语言处理（NLP）、隐马尔可夫模型（HMM）
✅ **蒙特卡洛方法（Monte Carlo）**：用于强化学习、图像生成
✅ **Bootstrap 方法**：用于估计统计量的分布
✅ **A/B 测试**：在互联网产品优化、广告投放中广泛应用

## **💻 实践练习**

### **1. 使用 Python 进行概率与统计计算**

📌 **推荐工具库**：

- `NumPy`（随机变量生成）
- `SciPy.stats`（概率分布计算）
- `Matplotlib / Seaborn`（数据可视化）

📌 **示例代码**：
📍 **生成正态分布数据并绘制直方图**

```
python复制编辑import numpy as np
import matplotlib.pyplot as plt
import seaborn as sns

# 生成 10000 个符合标准正态分布的随机数
data = np.random.normal(loc=0, scale=1, size=10000)

# 绘制直方图
sns.histplot(data, kde=True)
plt.title("Standard Normal Distribution")
plt.show()
```

📍 **计算置信区间**

```
python复制编辑import scipy.stats as stats

data = np.random.normal(loc=50, scale=10, size=100)
mean = np.mean(data)
confidence_interval = stats.norm.interval(0.95, loc=mean, scale=stats.sem(data))
print(f"95% 置信区间: {confidence_interval}")
```
